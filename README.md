# 🌿 EcoTrend - E-commerce Sustentável

## 📖 Sobre o Projeto

A **EcoTrend** é uma interface web front-end para um e-commerce especializado em produtos sustentáveis e ecológicos. O principal objetivo do projeto é promover um estilo de vida mais consciente, oferecendo uma experiência de navegação fluida, moderna e totalmente responsiva.

Este projeto foi desenvolvido como requisito de avaliação para o **Check-Point 04** da disciplina de Front-end Design, sob orientação do **Prof. Lucas Sousa** (Engenharia de Software - FIAP).

## ✨ Funcionalidades e Estrutura

* **Página Inicial (Home):** Carrossel de destaques em tela cheia e grid inteligente de produtos sustentáveis.
* **Página de Categorias:** Listagem de produtos em formato Grid com barra lateral estilizada para filtros (preço e marca).
* **Página de Produto Dinâmica:** Utilização de **JavaScript (Vanilla)** e parâmetros de URL para renderizar os detalhes técnicos, preços e imagens de cada produto dinamicamente em uma única página (`produto.html`), simulando o comportamento de um banco de dados e otimizando o código.
* **Página de Contato:** Formulário completo e estruturado para atendimento ao cliente.
* **Design Responsivo e Customizado:** Layout que se adapta perfeitamente a dispositivos móveis e desktops, com uma identidade visual única (gradientes e fontes customizadas) sobreposta ao Bootstrap.

## 🛠️ Tecnologias Utilizadas

* **HTML5 & CSS3** (Utilizando CSS Grid Layout e Flexbox).
* **Bootstrap 5** (Arquivos integrados localmente para garantir o funcionamento offline da Navbar, Cards e Carrossel).
* **JavaScript** (Lógica de roteamento dinâmico de produtos no front-end).
* **Google Fonts** (Uso da fonte *Comfortaa* no logotipo para reforçar a identidade visual premium).
* **Font Awesome** (Ícones vetoriais em todo o sistema).
